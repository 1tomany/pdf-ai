# PDF Extractor
This library extracts data from PDF files.

## Installation
```
composer require 1tomany/pdf-extractor
```

## Run Test Suite
``` bash
./vendor/bin/phpunit
```

## Run Static Analysis
``` bash
./vendor/bin/phpstan
```
